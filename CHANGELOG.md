# Changelog

All notable changes to Ursly Agent will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Multi-platform CI/CD builds (macOS, Windows, Linux)
- Comprehensive documentation (CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md)
- GitHub issue and pull request templates
- Enhanced README with latest features and best practices

### Changed

- Improved CI workflow for cross-platform testing
- Enhanced documentation structure
- Updated agents.md with comprehensive architecture guide

## [1.0.0] - 2024-12-25

### Added

- AI agent orchestration platform
- Keycloak OIDC authentication
- Entitlement-based authorization (RBAC, ABAC, ReBAC)
- Multi-model support (Ollama, OpenAI-compatible)
- Real-time WebSocket communication
- gRPC service for Ollama integration
- Audit logging and compliance features
- Task orchestration and scheduling
- Context window management
- Web UI and Desktop app (Tauri)
- OpenAPI/Swagger documentation

### Security

- JWT token-based authentication
- Organization-scoped multi-tenancy
- Comprehensive audit trails
- Secure credential management
